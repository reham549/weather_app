# 🌤️ Weather App

A simple and responsive Weather App built with React that allows users to search for any city and view current weather details such as temperature, condition, and wind speed.

The project focuses on using APIs, handling asynchronous data, and building clean reusable components with a modern UI.

## 🚀 Features
- Search for cities  
- Display weather data  
- Loading & error handling  
- Responsive design  

## 🛠️ Tech Stack
React • Axios • CSS • Bootstrap
